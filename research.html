QED-AI

Role: Creator and Lead Developer

A formal-mathematics proof assistant powered by machine learning. QED-AI pairs a decoder Transformer trained from scratch (≈32k vocabulary, 512-dimensional embeddings, 8 attention heads, 6 layers, 1024-token context) with a custom tokenizer to suggest proof steps, and verifies each generated proof by compiling it in Lean 4. It is served through a Dockerized web app on Hugging Face Spaces. The trained model weights and tokenizer are published as a separate Hugging Face model repository.

  • Source Code: GitHub Repository
  • Model Weights: Hugging Face Model
